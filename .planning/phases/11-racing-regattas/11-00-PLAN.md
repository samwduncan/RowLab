  <action>
Add new models to support Phase 11 requirements:

1. **Event model** (sits between Regatta and Race):
```prisma
model Event {
  id          String   @id @default(uuid())
  regattaId   String
  name        String   // "Varsity 8+", "2V8+", "JV4+", etc.
  category    String?  // Predefined or custom category
  scheduledDay Int?    // Day number for multi-day regattas (1, 2, 3)
  sortOrder   Int      @default(0)
  createdAt   DateTime @default(now())

  regatta Regatta @relation(fields: [regattaId], references: [id], onDelete: Cascade)
  races   Race[]

  @@index([regattaId])
  @@map("events")
}
```

2. Update **Race model** to reference Event instead of Regatta:
- Add `eventId String` field
- Add relation to Event
- Keep `regattaId` for now (computed via Event.regattaId for queries)

3. **ChecklistTemplate model**:
```prisma
model ChecklistTemplate {
  id        String   @id @default(uuid())
  teamId    String
  name      String   // "Home Regatta", "Away Regatta"
  isDefault Boolean  @default(false)
  createdAt DateTime @default(now())

  team  Team                 @relation(fields: [teamId], references: [id], onDelete: Cascade)
  items ChecklistTemplateItem[]

  @@index([teamId])
  @@map("checklist_templates")
}
```

4. **ChecklistTemplateItem model**:
```prisma
model ChecklistTemplateItem {
  id          String   @id @default(uuid())
  templateId  String
  text        String
  role        String   // "coach", "coxswain", "anyone"
  sortOrder   Int      @default(0)

  template ChecklistTemplate @relation(fields: [templateId], references: [id], onDelete: Cascade)

  @@map("checklist_template_items")
}
```

5. **RaceChecklist model** (instance for a specific race):
```prisma
model RaceChecklist {
  id         String   @id @default(uuid())
  raceId     String   @unique
  templateId String?
  createdAt  DateTime @default(now())

  race     Race                @relation(fields: [raceId], references: [id], onDelete: Cascade)
  template ChecklistTemplate?  @relation(fields: [templateId], references: [id])
  items    RaceChecklistItem[]

  @@map("race_checklists")
}
```

6. **RaceChecklistItem model**:
```prisma
model RaceChecklistItem {
  id           String    @id @default(uuid())
  checklistId  String
  text         String
  role         String
  completed    Boolean   @default(false)
  completedBy  String?   // User ID who completed
  completedAt  DateTime?
  sortOrder    Int       @default(0)

  checklist RaceChecklist @relation(fields: [checklistId], references: [id], onDelete: Cascade)

  @@map("race_checklist_items")
}
```

7. **ExternalRanking model**:
```prisma
model ExternalRanking {
  id              String   @id @default(uuid())
  teamId          String
  externalTeamId  String
  boatClass       String
  source          String   // "row2k", "usrowing", "regattacentral", "manual"
  ranking         Int
  season          String?
  updatedDate     DateTime
  notes           String?
  createdAt       DateTime @default(now())

  team         Team         @relation(fields: [teamId], references: [id], onDelete: Cascade)
  externalTeam ExternalTeam @relation(fields: [externalTeamId], references: [id])

  @@unique([teamId, externalTeamId, boatClass, source, season])
  @@map("external_rankings")
}
```

8. Update **Regatta model** with additional metadata fields:
- Add `endDate DateTime?` for multi-day regattas
- Add `host String?` for hosting organization
- Add `venueType String?` (lake, river, etc.)
- Add `externalUrl String?` for RegattaCentral/Row2k links
- Add `teamGoals String?` for pre-regatta notes

9. Add relations to existing models (Team, Regatta, Race, ExternalTeam).
