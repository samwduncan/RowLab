Create a custom hook for managing multi-step wizard state.

State:
- `step: number` - Current step (0-indexed, 0-3)
- `maxStepReached: number` - Highest step user has reached (for validation)
- `isSubmitting: boolean` - Submission in progress

Actions:
- `nextStep()` - Advance to next step (validates current step first)
- `prevStep()` - Go back one step
- `goToStep(n: number)` - Jump to specific step (only if <= maxStepReached)
- `reset()` - Reset to step 0

Step definitions:
```typescript
export const WIZARD_STEPS = [
  { id: 0, name: 'Session Info', description: 'Date, boat class, conditions' },
  { id: 1, name: 'Add Pieces', description: 'Create pieces with boats' },
  { id: 2, name: 'Assign Athletes', description: 'Place athletes in seats' },
  { id: 3, name: 'Review & Submit', description: 'Verify and create session' },
] as const;
```

The hook should be independent of react-hook-form - form state is managed separately in the wizard container.

Return type:
```typescript
{
  step: number;

  isSubmitting: boolean;
  steps: typeof WIZARD_STEPS;
  nextStep: () => void;
  prevStep: () => void;
  goToStep: (n: number) => void;
  setIsSubmitting: (v: boolean) => void;
  reset: () => void;
  isFirstStep: boolean;
  isLastStep: boolean;

}
```

Create the main wizard container that orchestrates steps.

Props:
- `onComplete: (data: SessionFormData) => void` - Called when wizard completes
- `onCancel?: () => void` - Called when user cancels
- `initialData?: Partial<SessionFormData>` - Pre-fill form data (for editing)

Structure:
```tsx
import { useForm, FormProvider } from 'react-hook-form';
import { zodResolver } from '@hookform/resolvers/zod';
import { sessionCreateSchema } from '@/v2/types/seatRacing';
import { useSessionWizard, WIZARD_STEPS } from '@/v2/hooks/useSessionWizard';

function SessionWizard({ onComplete, onCancel, initialData }) {
  const wizard = useSessionWizard();
  const methods = useForm({
    resolver: zodResolver(sessionCreateSchema),
    defaultValues: initialData || {
      date: new Date().toISOString().split('T')[0],
      boatClass: '',
      conditions: null,
      location: '',
      description: '',
      pieces: [],
    },
    mode: 'onChange', // Validate on change for step validation
  });

  const handleNext = async () => {
    // Validate current step fields before advancing
    const fieldsToValidate = getStepFields(wizard.step);
    const isValid = await methods.trigger(fieldsToValidate);
    if (isValid) wizard.nextStep();
  };

  const handleSubmit = methods.handleSubmit(async (data) => {
    wizard.setIsSubmitting(true);
    try {
      await onComplete(data);
    } finally {
      wizard.setIsSubmitting(false);
    }
  });

  // Step components (only Step 1 implemented in this plan)
  const stepComponents = [
    <SessionMetadataStep key="metadata" />,
    <div key="pieces">Step 2: Pieces (coming in Plan 04)</div>,
    <div key="assignments">Step 3: Assignments (coming in Plan 05)</div>,
    <div key="review">Step 4: Review (coming in Plan 06)</div>,
  ];

  return (
    <FormProvider {...methods}>
      <form onSubmit={handleSubmit} className="space-y-8">
        {/* Step indicator */}
        <StepIndicator
          currentStep={wizard.step}
          steps={WIZARD_STEPS}
          maxReachable={wizard.maxStepReached}
          onStepClick={wizard.goToStep}
        />

        {/* Current step content */}
        <div className="min-h-[300px]">
          {stepComponents[wizard.step]}
        </div>

        {/* Navigation buttons */}
        <div className="flex justify-between pt-6 border-t border-border">
          <div>
            {onCancel && (
              <button type="button" onClick={onCancel} className="btn-secondary">
                Cancel
              </button>
            )}
          </div>
          <div className="flex gap-3">
            {!wizard.isFirstStep && (
              <button type="button" onClick={wizard.prevStep} className="btn-secondary">
                Previous
              </button>
            )}
            {wizard.isLastStep ? (
              <button type="submit" disabled={wizard.isSubmitting} className="btn-primary">
                {wizard.isSubmitting ? 'Creating...' : 'Create Session'}
              </button>
            ) : (
              <button type="button" onClick={handleNext} className="btn-primary">
                Next
              </button>
            )}
          </div>
        </div>
      </form>
    </FormProvider>
  );
}
```

Helper function `getStepFields(step: number): string[]` returns fields to validate per step:
- Step 0: ['date', 'boatClass']
- Step 1: ['pieces'] (validated in Plan 04)
- Step 2: [] (assignments validated in Plan 05)
- Step 3: [] (review step, no validation)
